# Prehevil Treaty ETA website

> **Note:** This repository pertains to fictional countries on the [bits & Bytes Minecraft server](https://bnbmc.net).

This is the website for applying for electronic travel authorizations for the Prehevil Treaty free-travel zone.

It is split into a frontend and a backend.

## Frontend

The frontend uses TypeScript, Vue and Nuxt.

Node.js must be installed. The following commands can be run in the `frontend` directory.

To install dependencies:
```
npm install
```

To view the site during development with automatic reload:
```
npm run dev
```

To build the site:
```
npm run build
```

The output will be in `frontend/dist`.

To preview the site after building:
```
npm run preview
```
